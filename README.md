# Basic Chatbot - CodeAlpha Python Internship

## 📌 Project Description
This is a simple rule-based chatbot developed using Python as part of the CodeAlpha Python Programming Internship.

The chatbot interacts with users by responding to predefined messages such as greetings, basic questions, and exit commands.

## 🚀 Features
- Greets the user
- Responds to "hello", "hi", and "hey"
- Answers "How are you?"
- Tells its name
- Responds to "Thank you"
- Exits when the user types "bye"
- Handles unknown messages gracefully

## 🛠️ Technologies Used
- Python 3
- VS Code

## ▶️ How to Run
1. Install Python 3.
2. Open the project in VS Code.
3. Run the following command:

```bash
python basic_chatbot.py
```
